Skyrim Guide

This is a completionists guide for the video game Elder Scrolls V: Skyrim. Helping users track their progress to collect all items, visit all locations and speak to all the hundreds of characters. It also provides location information to help guide them on their path to a true 100%

The data for this app was collected from various fan sites using Selenium to save them into a CSV file which was used to create the MySQL database.

The app itself was created in C# using Xamarin Forms. I created the map by using the rough lat/long of items from one of the fan sites then by finding an appropriate map and splitting it up into sections I could highlight a section if the coordinates fell within.

I did attempt the map using an exact pin on the coordinates, but unfortunatley the locations I had gathered didn't seem to line up perfectly with the map.
